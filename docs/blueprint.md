# **App Name**: Hello, World!

## Core Features:

- Simple Greeting: Display a 'Hello, World!' message on the screen.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a calm and friendly feel.
- Background color: Very light gray (#F0F0F0) for a clean interface.
- Accent color: Soft green (#90EE90) for a subtle highlight.
- Font: 'Inter' (sans-serif) for a modern, neutral look in both headlines and body.
- Simple, centered layout with good spacing.
