# **App Name**: Hello, World!

## Core Features:

- Basic UI: Display a simple greeting message.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey a sense of peace and welcome.
- Background color: Very light blue (#E3F2FD), a lighter tint of the primary color, for a clean and airy feel.
- Accent color: A muted purple (#9575CD), chosen for being analogous to blue while offering a contrasting tone.
- Body and headline font: 'Inter', a grotesque-style sans-serif, chosen for its modern and neutral appearance suitable for both headlines and body text.
- Simple, line-based icons to maintain a clean aesthetic.
- Clean, minimalist layout with plenty of whitespace to avoid clutter.
- Subtle fade-in effect for the greeting message upon loading.
